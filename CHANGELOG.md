# Changelog

## 1.0.0

### Added
- Initial release of ProDialog
- DialogType: success, error, warning, info, question, custom
- 5 dialog entry animations (scale, fade, slideUp, slideDown, bounce)
- 5 icon animations (bounce, pulse, rotate, shake, none)
- Glassmorphism and gradient background support
- Auto-dismiss, loading indicator, close button
- Convenience helpers: showSuccessDialog, showErrorDialog, showWarningDialog, showLoadingDialog
- Full Semantics/accessibility support
- Zero external dependencies
